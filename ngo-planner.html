NGO Planner

AI-powered social media planning tool for NGOs

Overview

NGO Planner is a content planning tool designed for nonprofit organizations. It helps NGOs plan campaigns, generate AI-assisted content, manage social media calendars, and understand engagement through analytics.

My Role

Product Manager

  • Created the product vision, PRD, and user personas
  • Mapped user journeys & key features (MoSCoW prioritization)
  • Conducted competitor analysis and success metrics
  • Collaborated with the UI/UX team for wireframes and flows

Key Features

  • AI Content Generator
  • Template Library & Moodboard
  • Content Calendar & Scheduler
  • Campaign Performance Analytics

Outcome

Delivered a validated product concept with clear user needs, defined scope, and feature prioritization ready for MVP development.
